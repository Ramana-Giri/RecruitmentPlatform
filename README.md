## Overview
RecruitmentPlatform is intended to provide an end-to-end system for companies to:
- Create and manage job postings
- Accept and screen candidate applications
- Track interview stages and feedback
- Coordinate hiring teams and roles
- Generate hiring metrics and reports


## Features
- Job posting creation and management
- Candidate application intake (resume upload, cover letter)
- Interview scheduling and feedback collection
- Role-based access control (recruiter, interviewer)
- API-first design for integrations with ATS/HR systems

## Tech stack
- Backend: java Springboot Spring JPA
- Database: MySQL
- Frontend: ThymeLeaf 
- CI: GitHub

## Architecture  
High-level components:
- API Server — REST endpoints for jobs, candidates, interviews
- Database — persistent storage of users, jobs, applications, history
- Frontend — web UI for recruiters and hiring managers
